Shopping Store Web Application

A simple full functional shopping store web base application for users browsing and ordering cloths products. Users can do some actions, such as add items to shop cart, search cloths, rating and leave comments.

Techniques used: AngularJS, Node.JS, ExpressJS, MongoDB, HTML5 and CSS3.

Recipes Sharing Web Application

A food recipes sharing web base application for users creating and sharing recipes with photos; users can do actions, such as search, like, rate, make or delete comments.

Techniques used: EmberJS, Ember-Cli, Node.JS, MongoDB, HTML5 and CSS3.

World Clock Web Application

This world clock web base application is to imitate the iPhone Clock App. User can search a city in the world and add the clock to the board. It also have timer and stop watch functions.

Techniques used: EmberJS, Ember-Cli, Node.JS, MongoDB, HTML5 and CSS3.

Big Data Images Search Application

Implemented a Java Hadoop MapReduce program to find a kitty image among 10GB photos dataset by applying two mapper and refined P-hash algorithm to increase the accuracy, which is almost 50% correctness.

Techniques used: Apache Hadoop and Java
